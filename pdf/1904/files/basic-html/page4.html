           封面故事  ⃣丝路星空人的故事⃤俄罗斯摩尔曼斯克的星空出现了一簇翡翠极光，聚集的光束犹如碧桥，绽放的光束宛如孔雀开屏⃞
           捕捉美轮美奂的这一幕‚极光秀‛知名星空摄影师戴剑锋，他在重庆大学就读期间就是痴迷的天文爱好者，曾作为天文科普志愿者，
           随同⃣大学科普⃤编委会专家来到重庆市大足区复隆学校少年科学院开展天文科普活动，为孩子进行天文知识的科普⃞长期以来，
           虽然经历千辛万苦，但他的步履从未停止⃞他拍摄的绮丽星空图，凝聚着星空人携手‚一带一路‛的探索与发现⃞（撰文 / 刘洁嫚 )
